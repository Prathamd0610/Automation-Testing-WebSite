🔐 Authentication Flow

JWT issued as httpOnly cookies (access + refresh). The SPA auto‑refreshes on 401.

sequenceDiagram
    actor U as User
    participant C as React SPA
    participant A as API (Express)
    participant DB as MongoDB
    U->>C: Enter email + password
    C->>A: POST /api/auth/login
    A->>DB: Find user + bcrypt compare
    DB-->>A: User record
    A-->>C: 200 · httpOnly cookies · accessToken
    C->>A: GET /api/auth/me
    A-->>C: Current user
    C-->>U: Redirect to dashboard
    Note over C,A: On 401 the SPA calls POST /api/auth/refresh
